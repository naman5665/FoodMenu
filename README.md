# FoodMenu
A destination to find all the details of the food that you are eating from the menu.🍔☕️

The idea of this app is to give you the details of food before you order or before you eat😉

<h2><strong>Technical Details:</strong></h2>

_UI/UX_:- The UI of this app is made purely using Jetpack Compose(the new UI Android Toolkit).

_Animation is being used on main screen using compose animation library_

_Service_ We are obtaining the data from local json file. We are converting this json data to use in our codebase using GSON library

_MVVM architecture was followed throught the app_

_Material3 library is used throught the app to make the app more user friendly and integrate with new trending designs_
